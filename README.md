## Integrantes
- Dana Albreo
- Laura Bohada
- Eliana Rodríguez

## Descripción
Este proyecto consiste en una aplicación desarrollada en Python que permite administrar una agenda de películas.

El sistema permite:
- Consultar la película más larga
- Consultar la duración promedio
- Buscar películas de estreno
- Consultar películas clasificación 18+
- Reagendar películas
- Verificar si una persona puede ser invitada según su edad
